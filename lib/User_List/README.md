## 📌 Overview

**Flutter User List UI** is a simple and clean user interface module built with Flutter that displays a list of users in a structured and visually appealing way.

This project focuses on building reusable UI components for listing users, making it useful for applications like social apps, contact lists, or admin dashboards.

* Build reusable list components
* Understand structured data representation

## 🧠 How It Works (Easy Explanation)

This project demonstrates how Flutter handles list-based UI.

👉 Core idea:

* A list of users (data) is created
* Each user is displayed using a custom UI widget
* The list is rendered using `ListView`

### 🔄 Flow:

1. `main.dart` → App starts
2. Navigates to User List screen
3. User data is loaded (static/dummy)
4. `ListView.builder` generates UI items
5. Each item shows user details (name, info, etc.)

👉 Simple terms:

* Data → Loop → UI generate
