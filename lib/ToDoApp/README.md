## 📌 Overview

**Flutter ToDo App** is a simple and clean task management application built using Flutter.

This project demonstrates how users can create, manage, and track their daily tasks efficiently using a minimal and user-friendly interface.

It is perfect for beginners who want to understand Flutter UI + basic app logic together.

---

## 🎯 Project Goals

* Learn Flutter UI + state handling
* Build a real-life useful application
* Understand task management logic
* Practice clean UI design

---

## ✨ Key Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 📋 Display task list
* ⚡ Instant UI update

---

## 🧠 How It Works (Easy Explanation)

This app works based on a simple **task list system**.

### 🔄 Flow:

1. User opens the app
2. Adds a new task using input field
3. Task gets added to a list
4. UI updates automatically
5. User can:

   * Mark task as complete ✅
   * Delete task ❌

👉 Internally:

* Tasks are stored in a list
* Each task has:

  * Title
  * Completion status

👉 Flutter widgets rebuild when state changes
