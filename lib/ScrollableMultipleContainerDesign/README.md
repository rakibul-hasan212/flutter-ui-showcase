## 📌 Overview

**Scrollable Multiple Container Design** is a Flutter UI implementation that demonstrates how to create a scrollable layout containing multiple styled containers.

This project focuses on building smooth scrolling interfaces and organizing multiple UI elements vertically in a clean and responsive way.

It is especially useful for beginners who want to understand how scrolling works in Flutter apps.

## 🧠 How It Works (Simple Explanation)

This UI design is built using Flutter’s core layout widgets.

### 🔄 Flow:

1. `main.dart` → App starts here
2. Loads the main screen widget
3. A scrollable widget is used (like `SingleChildScrollView` or `ListView`)
4. Inside it → multiple `Container` widgets are placed
5. Each container represents a UI section

---

### 🧩 Key Widgets Used:

* `SingleChildScrollView` → Enables scrolling
* `Column` → Stacks containers vertically
* `Container` → Creates UI blocks
* `Padding / Margin` → Adds spacing
* `BoxDecoration` → Styling (color, border, radius)

---

👉 So basically:

* Many containers → inside Column
* Column → inside ScrollView
* ScrollView → makes everything scrollable
