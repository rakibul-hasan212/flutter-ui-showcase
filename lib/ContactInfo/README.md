## 📌 Overview

This project demonstrates how to build a **Contact Information UI using ListView in Flutter**.

It showcases how multiple contact items can be displayed in a scrollable list format, making it useful for apps like contact lists, user directories, or messaging apps.

---

## 🎯 Project Goals

* Learn how to use `ListView` in Flutter
* Display structured data (contact info) in UI
* Build clean and reusable UI components
* Understand scrolling layouts

---

## ✨ Key Features

* 📋 Display multiple contacts in a list
* 🔽 Scrollable UI using `ListView`
* 👤 Contact info (name, phone, etc.) display
* 🎨 Clean and simple UI layout
* ♻️ Reusable widget structure

---

## 🧠 How It Works

This project uses Flutter’s **ListView widget** to render a list of contact items.

### 🔄 Flow Explanation:

1. A list of contact data is defined (e.g., name, phone number)
2. `ListView` iterates through the data
3. Each item is displayed using a custom layout (Row / Column)
4. The UI becomes scrollable automatically

👉 Each contact item typically includes:

* Name
* Phone number
* Optional icons (call/message)
