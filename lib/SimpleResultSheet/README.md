## 📌 Overview

**Simple Result Sheet UI** is a Flutter-based UI component that displays a student's result sheet in a clean and structured format.

This module demonstrates how to design a result/score screen using Flutter widgets, making it ideal for educational apps or student management systems.

---

## 🎯 Project Goals

* Build a clean result display UI
* Practice layout structuring in Flutter
* Learn how to present structured data
* Create reusable UI components

---

## ✨ Key Features

* 📊 Student result display layout
* 🧾 Subject-wise marks presentation
* 📈 Total / summary section
* 🎨 Clean and readable UI design
* 📱 Mobile-friendly layout

## 🧠 How It Works (Easy Explanation)

This project focuses only on **UI design and data presentation**.

👉 Flow of the app:

1. `main.dart` → App starts
2. Loads the Result Screen
3. Displays:

   * Student info (name, class, etc.)
   * List of subjects with marks
   * Total / result summary

👉 Internally:

* `Column` → Vertical layout
* `Row` → Align subject & marks
* `Container` → Styling (padding, color, spacing)
* `Text` → Display data

👉 No backend, everything is static/demo data.
